# Practical Testing

요구사항

키오스크 주문을 위한 상품 후보 리스트 조회하기
상품의 판매 상태 : 판매중, 판매보류, 판매중지
-> 판매중, 판매보류인 상태의 상품을 화면에 보여준다.

상품 정보
id, 상품 번호, 상품 타입, 판매 상태, 상품 이름, 가격
